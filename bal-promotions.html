🎉 Programme de la soirée

  • 19h00 : Accueil des élèves et apéritif

📋 Informations pratiques

📊 Administration

Gestion des inscriptions via Google Sheets

📋 Accéder aux inscriptions

Toutes les inscriptions sont enregistrées dans un Google Sheet partagé. Vous pouvez y consulter, filtrer et gérer les inscriptions directement.

📊 Ouvrir Google Sheets

Remplacez ce lien par l'URL de votre Google Sheet dans le code

ℹ️ Fonctionnalités du Sheet

  • Onglet "Élèves" : toutes les inscriptions élèves
  • Onglet "Bénévoles" : toutes les inscriptions bénévoles
  • Ajoutez une colonne "Fiche reçue" pour le suivi
  • Utilisez les filtres pour trier par classe, statut, etc.
  • Partagez le Sheet avec le secrétariat
💡 Astuce : Créez des règles de mise en forme conditionnelle dans Google Sheets pour colorer automatiquement les lignes selon le statut (fiche reçue ou non).
